Over- and Undersampling

For unevenly distributed classes in the data, it might be useful to oversample the minority or undersample the majority class.

data.frame

First we show how it is done for data.frames:

library(mlr)
classA = data.frame(x = rnorm(20, mean=1), class="A")
classB = data.frame(x = rnorm(80, mean=2), class="B")
data = rbind(classA, classB)
table(data$class)

## 
##  A  B 
## 20 80

data.over = oversample(data, "class", rate=4)

## Error: unused argument ("class")

table(data.over$class)

## Error: object 'data.over' not found

data.under = undersample(data, "class", rate=1/4)

## Error: unused argument ("class")

table(data.under$class)

## Error: object 'data.under' not found

Tasks

Now let's see what the effect is for a classification task:

data.even = rbind(
  data.frame(x = rnorm(50, mean=1), class="A"),
  data.frame(x = rnorm(50, mean=2), class="B"))
task = makeClassifTask(data=data, target="class")
task.over = oversample(task, rate=4)
task.under = undersample(task, rate=1/4)
lrn = makeLearner("classif.PART")
mod = train(lrn, task)

## Error: cannot coerce class "c("classif.PART", "RLearnerClassif",
## "RLearner", "Learner")" to a data.frame

mod.over = train(lrn, task.over)

## Error: cannot coerce class "c("classif.PART", "RLearnerClassif",
## "RLearner", "Learner")" to a data.frame

mod.under = train(lrn, task.under)

## Error: cannot coerce class "c("classif.PART", "RLearnerClassif",
## "RLearner", "Learner")" to a data.frame

performance(predict(mod, newdata = data.even), measures=mmce)

## Error: error in evaluating the argument 'object' in selecting a method for function 'predict': Error: object 'mod' not found

performance(predict(mod.over, newdata = data.even), measures=mmce)

## Error: error in evaluating the argument 'object' in selecting a method for function 'predict': Error: object 'mod.over' not found

performance(predict(mod.under, newdata = data.even), measures=mmce)

## Error: error in evaluating the argument 'object' in selecting a method for function 'predict': Error: object 'mod.under' not found

Learners

Note that it is highly dependent on the learner if over- or undersampling will be useful. You are also able to wrap the over- and undersampling within the learner.

lrn.over = makeOversampleWrapper(lrn, osw.rate=4)
lrn.under = makeUndersampleWrapper(lrn, usw.rate=1/4)
mod = train(lrn, task)

## Error: cannot coerce class "c("classif.PART", "RLearnerClassif",
## "RLearner", "Learner")" to a data.frame

mod.over = train(lrn.over, task)

## Error: cannot coerce class "c("OversampleWrapper", "BaseWrapper",
## "Learner")" to a data.frame

mod.under = train(lrn.under, task)

## Error: cannot coerce class "c("UndersampleWrapper", "BaseWrapper",
## "Learner")" to a data.frame

performance(predict(mod, newdata = data.even), measures=mmce)

## Error: error in evaluating the argument 'object' in selecting a method for function 'predict': Error: object 'mod' not found
