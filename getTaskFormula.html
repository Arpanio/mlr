getTaskFormula {mlr}R Documentation

Get formula of a task.

Description

This is simply the target ~ . formula.

Usage

  getTaskFormula(x, delete.env = TRUE)

Arguments

x

[SupervisedTask | TaskDesc]
Task or its description object.

delete.env

[delete.env]
Delete enviroment attached to returned formula? Don't ask why this option exists, R sucks. Default is TRUE.

Value

[formula].

Examples


task <- makeClassifTask(data = iris, target = "Species")
getTaskFormula(task)

## Species ~ .
## NULL
