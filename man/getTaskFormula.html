getTaskFormula {mlr}R Documentation

Get formula of a task.

Description

This is simply the target ~ . formula.

Usage

getTaskFormula(x, target = getTargetNames(x), env = NULL)

Arguments

x

[SupervisedTask | TaskDesc]
Task or its description object.

target

[character(1)]
Left hand side of formula. Default is defined by task x.

env

[environment]
Environment of the formula. Set this to parent.frame() for the default behaviour. Default is NULL which deletes the environment.

Value

[formula].

Examples


task <- makeClassifTask(data = iris, target = "Species")
getTaskFormula(task)

## Species ~ .
## NULL
