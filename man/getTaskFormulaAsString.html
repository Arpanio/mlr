getTaskFormulaAsString {mlr}R Documentation

Get formula of a task as a string.

Description

This is simply “<target> ~ .”.

Usage

  getTaskFormulaAsString(x)

Arguments

x

[SupervisedTask | TaskDesc]
Task or its description object.

Value

[character(1)].

Examples


task <- makeClassifTask(data = iris, target = "Species")
getTaskFormulaAsString(task)
