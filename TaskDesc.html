TaskDesc {mlr}R Documentation

Description object for task.

Description

Description object for task, encapsulates basic statistics without having to store the complete data set.

Details

Object members:

id [character(1)]

Id string of task.

type [character(1)]

type Type of task, either “classif” for classification or “regr” for regression.

target [character(1)]

Name of target variable.

size[integer(1)]

Number of cases.

n.feat [integer]

Number of features, named vector with entries: “numerics”, “factors”.

class.levels [character]

All possible classes. NA if not classification.

has.missings [logical(1)]

Are missing values present?

has.blocking [logical(1)]

Is blocking available in task for observations?

positive [character(1)]

Positive class label for binary classification, NA else.

negative [character(1)]

Negative class label for binary classification, NA else.
